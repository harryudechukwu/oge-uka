# Oge Uka

Daily Catholic Mass readings in English and Igbo, presented side by side in a clean, bilingual Progressive Web App.

Read today's Mass readings: First Reading, Responsorial Psalm, Alleluia, and Gospel, with a simple toggle between English and Igbo. Works offline once loaded.

[Live site](https://ogeuka.netlify.app/)

Built with Next.js, Tailwind CSS, and served via Netlify.
